## How to run a real calibration

```
SUPABASE_URL=... SUPABASE_SERVICE_KEY=... \
node .github/scripts/calibrate-image-weights.js
```

This will overwrite this file with a results table and a recommended split
— but per MODULE 4's own rule ("do not guess, measure"), the weight
constants in `hf-analyze.ts` should only be changed by hand after reading
that output, not automatically.
